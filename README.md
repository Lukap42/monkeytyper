# Monkey Typer
Monkey typer game written in C++ utilizing SMFL library for GUI elements

The game features:
- Customizable fonts (selectable in options) and font sizes (+/- keys in settings)
- Save and Load option (S and L keys)

## Used technologies
- SFML (Simple and Fast Multimedia Library) was used for creating the graphical elements
